pade.common.fix_newlines(msg)[source]¶

Attempt to wrap long lines as paragraphs.

Treats each line in the given message as a paragraph. Wraps each paragraph to avoid long lines. Returns a string that contains all the wrapped paragraphs, separated by blank lines.

pade.stat¶

Low-level statistical methods.

This module should be general-purpose, and not have any dependencies on the data model used in Pade or the workflow. The idea is that we may use these functions outside of the standard Pade workflow.

class pade.stat.FtestSqrt(layout_full, layout_reduced)[source]¶

Statistic that gives the square root of the f-test.

This is a simple wrapper around the Ftest that returns its square root. You can use this to simulate a t-test between two groups.

class pade.stat.Ttest(alpha)[source]¶

Deprecated since version 1.0.

This is the original Ttest implementation, but it probably doesn’t actually work with the current Pade, since it expects a statistic that takes all the groups and returns one statistic per feature. We may restore this in the future.

pade.stat.bins_custom(num_bins, stats)[source]¶

Get an array of bin edges based on the actual computed statistic values. stats is an array of length n. Returns an array of length num_bins + 1, where bins[m, n] and bins[m + 1, n] define a bin in which to count features for condition n. There is a bin edge for negative and positive infinity, and one for each statistic value.

pade.stat.bins_uniform(num_bins, stats)[source]¶

Returns a set of evenly sized bins for the given stats.

Stats should be an array of statistic values, and num_bins should be an integer. Returns an array of bin edges, of size num_bins + 1. The bins are evenly spaced between the smallest and largest value in stats.

Note that this may not be the best method for binning the statistics, especially if the distribution is heavily skewed towards one end.

pade.stat.cumulative_hist(values, bins)[source]¶

Create a cumulative histogram for values using the given bins.

The shape of values and bins must be the same except for the last dimension. So np.shape(values)[:-1] must equal np.shape(bins[:-1]). The last dimension of values is simply a listing of values. The last dimension of bins is the list of bin edges for the histogram.

pade.stat.cumulative_hist_shape(bins)[source]¶

Returns the shape of the histogram with the given bins.

The shape is similar to that of bins, except the last dimension has one less element.

pade.stat.random_orderings(full, reduced, R)[source]¶

Get an iterator over at most R random index shuffles.

Parameters:
  • full – the layout
  • reduced – the reduced layout
  • R – the maximum number of orderings to return
Returns:

iterator over random orderings of indexes

Each item in the resulting iterator will be an ndarray of the indexes in the given layouts. The indexes within each group of the reduced layout will be shuffled.
