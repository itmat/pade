PaGE_5.1 Documentation - the --num_bins command

Because PaGE cannot do infinite computation, it must granularize the range of the statistic. It divides the range into 1000 bins by default. Use this command to change the number of bins. This is unlikely to affect your results and probably you will not need to change this form the default.

This command takes one argument of type integer, which must be greater than 2.

Example: 

> perl PaGE_5.1.6.pl --num_bins 2000
