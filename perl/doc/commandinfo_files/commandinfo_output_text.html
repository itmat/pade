PaGE_5.1 Documentation - the --output_text command

Use this option to output the results also in tab delimited TEXT format. HTML results are also always output.

Note: The name of the TEXT file is the same as the name of the HTML file, except the suffix is changed to .txt.

This command takes no arguments.

Example: 

> perl PaGE_5.1.6.pl --output_text
