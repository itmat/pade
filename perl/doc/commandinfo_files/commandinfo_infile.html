PaGE_5.1 Documentation - the --infile command

Use this command to give the name of the file which contains the data. The data file must conform to the required format.

This command takes one argument of type string.

Example: This includes the file named "my_data". 

> perl PaGE_5.1.6.pl --infile my_data
