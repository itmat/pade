PaGE_5.1 Documentation - the --use_unlogged_data command

When using the t-statistic you can apply it either to the unlogged data, or to the logged data.

See the command --use_logged_data for more information on this.

Use this command to indicate you want to run on the unlogged data.

This command takes no arguments

Example: 

> perl PaGE_5.1.6.pl --use_unlogged_data
