PaGE_5.1 Documentation - the --outfile command

Use this command to give the name of the outfile.

The output will consist of a main HTML file and one or more auxiliary files. The name you specify will be used as the prefix for all of these files. It is not necessary to suffix your outfile name with ".html" or any other such suffix, the program will add the appropriate suffixes.

This command takes one argument of type string.

Note: If you do not specify this on the command line, the output file name will be derived from the input file name.

Example: This writes to the file named "my_results.html". 

> perl PaGE_5.1.6.pl --outfile my_results
