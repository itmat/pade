PaGE_5.1 Documentation - the --data_is_logged command

Use this option to indicate that the date file contains logged data.

This command takes no arguments.

Note 1: In the case of 2-channel data then this indicates the data file contains log ratios, or what are typically called "M" values.
Note 2: It doesn't matter to the algorithm what the base of the logarithm is.

Example: 

> perl PaGE_5.1.6.pl --data_is_logged
