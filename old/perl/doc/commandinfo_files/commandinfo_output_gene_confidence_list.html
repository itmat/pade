PaGE_5.1 Documentation - the --output_gene_confidence_list command

Every gene has some confidence of up- or down-regulation. This may not be output on the HTML file since that only reports confidences above the --level_confidence parameter. To obtain a file which maps all genes to their respective confidences, use this option.

This command takes no arguments.

Example: 

> perl PaGE_5.1.6.pl --output_gene_confidence_list
