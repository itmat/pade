PaGE_5.1 Documentation - the --aux_page_size command

This option specifies the minimum number of tags there can be in one pattern before the results for that pattern are written to an auxiliary page. This keeps the main results page from getting too large when there are thousands of genes in the report.

This command takes one argument of type integer, which must be greater than 10.

Note: The default aux_page_size is 500.

Example: 

> perl PaGE_5.1.6.pl --aux_page_size 100
