PaGE_5.1 Documentation - the --id_filter_file command

Use this command to give the name of a file which contains ID's, one per line. If this file is specified with the --id_filter_file command then only the rows in your data file with these ID's will be considered. This is useful in case you want to look at various subsets of genes and do not want to reparse your entire data file.

This command takes one argument of type string.

Note: It is important to be careful about the subsets used as it can bias the confidence measures.

  • It is okay to choose a subest based on biological function or some other biological characterization.
  • It is not appropriate to choose a subest of genes based on some statistical criteria, such as only genes with small variance, or only genes which show differential expression.

Example: This includes the file named "my_ids". 

> perl PaGE_5.1.6.pl --id_filter_file my_ids
Example of what the file my_ids might look like: 
G1
G4
G15
G33
G97
...
