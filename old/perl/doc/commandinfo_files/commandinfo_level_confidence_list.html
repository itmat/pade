PaGE_5.1 Documentation - the --level_confidence_list command

If you have more than two conditions, so you will have patterns of length two or longer, then you can set the level_confidence separately for each position in the pattern.

The level_confidence_list is a comma delimited list of confidences, one for each position in the pattern. Do not include any white-space in this list.

Note: If you have more than two conditions and you use --level_confidence then that confidence will be applied to all positions in the pattern.

This command takes one argument of type string, which must be a comma separated list of floats between 0 and 1.

Example: 

> perl PaGE_5.1.6.pl --level_confidence_list .8,.6,.95
