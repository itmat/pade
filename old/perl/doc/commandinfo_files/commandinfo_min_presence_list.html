PaGE_5.1 Documentation - the --min_presence_list command

This is similar to --min_presence, but it allows you to set the min_presence separately for each condition.

This command takes one argument of type string, which must be a comma delimited list of positive integers each greater than or equal to 2.

Note: If you set this to be higher than actual the number of replicates in some group, the program will complain.

Example: If there are four conditions, including the reference condition zero, then this parameter might be set as: 

 perl PaGE_5.1.6.pl --min_presence_list 4,3,5,2
